# EducationCenter API

Учебный backend-проект для управления образовательным центром.

API позволяет работать со студентами, группами, преподавателями,
курсами и предметами.

## Стек

- C#
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- Swagger / OpenAPI
- xUnit
- EF Core InMemory для тестов

## Возможности

- CRUD для основных сущностей
- Связи one-to-many и many-to-many
- DTO для запросов и ответов
- Проверка связанных сущностей
- Корректные HTTP-статусы
- Добавление и удаление связей
- Автоматические тесты

## Основные сущности

- `Course` — учебный курс
- `Subject` — предмет
- `Teacher` — преподаватель
- `Group` — учебная группа
- `Student` — студент

Курсы связаны с предметами, преподаватели — с предметами, группы — с
курсами, преподавателями и предметами, а студенты могут состоять в группах.

## Запуск проекта

Перед запуском необходимо создать базу данных PostgreSQL и проверить строку
подключения `DefaultConnection` в `EducationCenter.Api/appsettings.json`.

```bash
dotnet run --project EducationCenter.Api
```

В режиме разработки Swagger UI доступен по адресу приложения, указанному в
`EducationCenter.Api/Properties/launchSettings.json`.

## Тесты

Запустить все тесты решения можно командой:

```bash
dotnet test
```

Тесты проверяют основные операции контроллеров, работу связей, обработку
несуществующих внешних ID и защиту от удаления используемых сущностей.
